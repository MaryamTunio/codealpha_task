This repository contains 3 Python projects completed as part of the CodeAlpha Internship. Each project demonstrates core Python concepts such as loops, conditionals, file handling, and user interaction.

_Projects Included

---

** Chatbot

 Description
A simple rule-based chatbot that responds to user inputs using predefined answers.

 Features
 Interactive conversation
 Predefined responses

**Hangman Game
Description

A word guessing game where the user tries to guess the hidden word letter by letter within limited attempts.

Features
Random word selection
Letter guessing system
Limited attempts
Win/Lose conditions

** Stock Portfolio Tracker
Description

A program that calculates total investment value based on selected stocks and quantities.

Features
Predefined stock prices
User input for stocks and quantity
Portfolio value calculation
