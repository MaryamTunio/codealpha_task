# Chatbot Project 

##  Description
This is a simple Python-based chatbot that responds to basic user inputs. It runs in the terminal and continues the conversation until the user types "bye".

---

##  Features
- Greets the user
- Responds to basic questions
- Case-insensitive input handling
- Exit command ("bye")

---

## 🧠 How it works
The chatbot uses a `while loop` to continuously take user input and responds using `if-elif-else` conditions.

---

## ▶️ How to Run
```bash
python chatbot.py
