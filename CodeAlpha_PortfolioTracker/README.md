 Stock Portfolio Tracker


DESCRIPTION:The Stock Portfolio Tracker is a simple Python console-based application that allows users to track their stock investments

 Features
Add multiple stocks with quantities
Predefined stock price list
Calculates total investment value
Displays portfolio summary in console
Saves results to a file (portfolio.txt)
 How It Works
The program stores stock prices in a dictionary.
The user enters stock symbols and quantities.
The program checks if the stock exists.
It calculates:
Individual stock value
Total investment value
Results are displayed and saved in a text file.
💻 Technologies Used
Python 3
File Handling
Dictionaries
Loops & Conditional Statements
